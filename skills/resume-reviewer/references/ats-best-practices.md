# ATS (Applicant Tracking System) Best Practices

ATS systems are used by companies to filter and rank resumes before a human sees them.

## Formatting Guidelines
- **Simple Layout:** Use a single-column layout. Avoid complex tables or columns which can break the reading order.
- **Standard Fonts:** Use common fonts like Arial, Helvetica, or Calibri.
- **File Format:** PDF is usually preferred, but ensures the text is selectable (not an image).
- **Headers/Footers:** Avoid putting critical info (like contact info) in headers/footers, as some ATS skip them.

## Content Optimization
- **Keywords:** Include exact keywords from the JD.
- **Contextual Keywords:** Use related terms (e.g., if the JD asks for "Python," also mention "Django" or "Flask" if applicable).
- **Acronyms:** Use both the acronym and the full term (e.g., "Quality Management System (QMS)").
- **Standard Headings:** Use "Work Experience," "Education," and "Skills" rather than creative headers.
