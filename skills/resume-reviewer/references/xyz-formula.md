# XYZ Formula for Resume Bullets

The XYZ formula, popularized by Google, helps quantify your impact.

**Structure:**
"Accomplished [X] as measured by [Y], by doing [Z]."

- **X (Accomplished):** Start with a strong action verb and the result.
- **Y (Measured by):** Provide a metric or scale (percentage, revenue, users, time).
- **Z (By doing):** Describe the specific action or skill used to achieve X.

**Example:**
*   **Weak:** "Fixed bugs in the system."
*   **Strong (XYZ):** "Improved system stability (X) by 25% (Y) by implementing a new automated regression testing suite using Pytest (Z)."
