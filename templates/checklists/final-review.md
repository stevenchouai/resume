# AI Resume Tailoring Checklist

Use this checklist before committing or sending a tailored resume.

## Truthfulness

- [ ] No invented experience
- [ ] No invented metrics
- [ ] No hidden exaggeration from AI rewriting
- [ ] Every claim can be explained in an interview

## JD alignment

- [ ] Top 5 JD requirements are represented
- [ ] Important keywords are included naturally
- [ ] Resume title / positioning matches the role
- [ ] Strongest evidence appears in the top half of the page

## Bullet quality

- [ ] Bullets start with strong verbs
- [ ] Bullets include impact, scale, or constraints
- [ ] At least some bullets use the XYZ formula
- [ ] No vague phrases like “responsible for” unless necessary

## ATS / formatting

- [ ] PDF text is selectable
- [ ] No broken LaTeX symbols
- [ ] No tables that scramble reading order, unless tested
- [ ] File is one page or intentionally longer
- [ ] Filename is professional

## Privacy

- [ ] Private email / phone / address not committed unintentionally
- [ ] Company-specific JD not committed unless public and intended
- [ ] Analysis notes do not expose private strategy
- [ ] Generated PDFs are not committed unless deliberately sanitized
